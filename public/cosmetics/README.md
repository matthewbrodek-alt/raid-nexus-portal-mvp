# Cosmetics assets

This folder is reserved for account customization assets.

Recommended structure:

- `avatars/{status}/...` - optional preset avatar images.
- `avatar-frames/{status}/...` - transparent PNG/WebP frame overlays.

Status folders:

- `bronze`
- `silver`
- `gold`
- `platinum`
- `admin`

Keep transparent frame files square, for example `512x512` or `1024x1024`.
