Put original social icons in this folder.

Use transparent PNG files, ideally 256x256 or 512x512.

Required filenames:

```text
telegram.png
vk-video.png
vk-community.png
youtube.png
twitch.png
```

The site reads them from:

```text
/icons/social/telegram.png
/icons/social/vk-video.png
/icons/social/vk-community.png
/icons/social/youtube.png
/icons/social/twitch.png
```

If you only have SVG files, export them to PNG with the exact filenames above.
